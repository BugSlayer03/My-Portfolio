# 🌐 Personal Portfolio Website

This is a fully responsive personal portfolio website built to showcase my skills, projects, and contact information. It is designed with a clean UI and smooth user experience, adapting seamlessly across different screen sizes.

## 🚀 Features

* Responsive design for desktop, tablet, and mobile
* Smooth scrolling navigation
* Interactive UI with hover effects
* Projects showcase with GitHub links
* Contact section with external links (LinkedIn, GitHub, WhatsApp, Email)
* Clean and structured layout

## 🛠️ Tech Stack

* **HTML5** – Structure of the website
* **CSS3** – Styling, layout, and responsiveness
* **JavaScript** – Dynamic behavior (typewriter effect, interactions)

## 📸 Sections

* **Home** – Introduction with typewriter effect
* **About** – Brief overview of background and interests
* **Skills** – Technologies and tools
* **Projects** – Showcased work with repository links
